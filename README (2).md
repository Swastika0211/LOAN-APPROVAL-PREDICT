# 🏦 Loan Approval Predictor — ML + Streamlit App

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io)

An end-to-end machine learning project that predicts loan approvals using XGBoost (~98% accuracy), with a full Streamlit web interface.

---

## 📁 Repository Structure

```
loan-approval-predictor/
├── app.py                    # Streamlit application
├── loan_approval_ml.ipynb    # Google Colab notebook (EDA + ML training)
├── requirements.txt          # Python dependencies
├── loan_model.pkl            # Trained XGBoost model  ← generated by Colab
├── scaler.pkl                # StandardScaler         ← generated by Colab
├── feature_names.pkl         # Feature name list      ← generated by Colab
└── README.md
```

---

## 🚀 Step-by-Step Deployment Guide

### Step 1 — Train the Model in Google Colab

1. Open [Google Colab](https://colab.research.google.com)
2. Upload `loan_approval_ml.ipynb`
3. Upload your dataset `loan_approval_dataset.csv` when prompted
4. Run all cells (Runtime → Run all)
5. The last cell will **automatically download** 3 files:
   - `loan_model.pkl`
   - `scaler.pkl`
   - `feature_names.pkl`

### Step 2 — Set Up GitHub Repository

1. Create a new GitHub repository (e.g., `loan-approval-predictor`)
2. Upload all these files to the repo:
   ```
   app.py
   requirements.txt
   README.md
   loan_approval_ml.ipynb
   loan_model.pkl          ← downloaded from Colab
   scaler.pkl              ← downloaded from Colab
   feature_names.pkl       ← downloaded from Colab
   ```

### Step 3 — Deploy to Streamlit Cloud

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with your GitHub account
3. Click **"New app"**
4. Configure:
   - **Repository**: `your-username/loan-approval-predictor`
   - **Branch**: `main`
   - **Main file path**: `app.py`
5. Click **"Deploy!"**
6. ✅ Your app will be live in ~2 minutes!

## 📊 Features Used

| Feature | Description |
|---------|-------------|
| `cibil_score` | Credit score (300–900) — most important |
| `income_annum` | Annual income |
| `loan_amount` | Requested loan amount |
| `loan_term` | Loan duration in years |
| `total_assets` | Sum of all 4 asset types |
| `loan_to_income_ratio` | Engineered: loan ÷ income |
| `asset_to_loan_ratio` | Engineered: total assets ÷ loan |
| `income_per_dependent` | Engineered: income ÷ (dependents + 1) |
| `education` | Graduate / Not Graduate |
| `self_employed` | Yes / No |
| `no_of_dependents` | 0 to 5 |
| `residential_assets_value` | Property value |
| `commercial_assets_value` | Business property value |
| `luxury_assets_value` | Luxury items value |
| `bank_asset_value` | Bank holdings |
